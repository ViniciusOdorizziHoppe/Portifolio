🔒 ZAP AI — Política de Privacidade

Última atualização: Julho 2026

1. Dados Coletados

A extensão ZAP AI coleta apenas os dados estritamente necessários para seu funcionamento:

  • API Key: Sua chave da OpenAI/DeepSeek, armazenada localmente no navegador via chrome.storage.local. NUNCA é enviada para nossos servidores.
  • Preferências: Templates salvos, configurações de auto-sugestão, contagem diária de transcrições — tudo armazenado localmente.
  • Áudio de WhatsApp: Enviado diretamente para a API OpenAI Whisper para transcrição. Não armazenamos, não retemos, não compartilhamos.
  • Mensagens de WhatsApp: Apenas as últimas 5 mensagens são enviadas para a API DeepSeek para gerar sugestões de resposta. Não armazenamos histórico.

2. O que NÃO coletamos

  • Não lemos suas mensagens completas do WhatsApp
  • Não acessamos sua lista de contatos
  • Não coletamos dados de navegação
  • Não compartilhamos dados com terceiros
  • Não usamos cookies ou rastreadores

3. Armazenamento

Todos os dados são armazenados localmente no seu navegador usando chrome.storage.local. Nada é enviado para servidores externos além das APIs que você configura (OpenAI Whisper e DeepSeek).

4. Serviços de Terceiros

A extensão se conecta às seguintes APIs de terceiros, somente quando você as configura com sua própria chave:

  • OpenAI Whisper API (api.openai.com) — para transcrição de áudio
  • DeepSeek API (api.deepseek.com) — para geração de respostas IA

O uso dessas APIs está sujeito às políticas de privacidade da OpenAI e DeepSeek, respectivamente.

5. Seus Direitos

Como todos os dados ficam no seu navegador, você pode:

  • Limpar os dados a qualquer momento nas configurações da extensão
  • Desinstalar a extensão para remover todos os dados
  • Solicitar informações sobre dados armazenados (não há nada em servidores externos)

6. Pagamentos (Plano Pro)

O plano Pro é processado via Stripe. A Stripe coleta dados de pagamento conforme sua própria política de privacidade. A extensão não tem acesso aos seus dados de cartão de crédito.
